実行方法（実行ファイルの動かし方）

実行するためのコマンドは以下の通りである。

java -jar Forest_Program.jar

Mac OS Xの場合には、アプリケーションForest_Program.appをダブルクリックするほうが簡便である。ただし、今回は作成することが出来なかった。

また、openコマンドでForest_Program.appを開いても実行可能である。ただし、今回は作成することが出来なかった。

open Forest_Program.app
